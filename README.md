# 🎲 Sorteador de Números

Um sorteador de números simples e interativo desenvolvido com HTML, CSS e JavaScript.

O projeto permite que o usuário defina um intervalo de números e realize um sorteio aleatório de forma rápida e prática.

🔗 Demo online: https://joaojnr10.github.io/Sorteador-de-numeros/

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## 🎯 Funcionalidades

- Definir número inicial e número final
- Gerar número aleatório dentro do intervalo
- Interface simples e responsiva
- Funciona em computadores, tablets e celulares

## 💡 Objetivo do projeto

Este projeto foi desenvolvido como parte dos meus estudos em desenvolvimento front-end, com o objetivo de praticar manipulação do DOM, lógica de programação e criação de interfaces responsivas.
